## Завдання 7.10
Напишіть програму мовою C для створення послідовності випадкових чисел з плаваючою комою у діапазонах:
* (a) від 0.0 до 1.0
* (b) від 0.0 до n, де n — будь-яке дійсне число з плаваючою точкою.
Початкове значення генератора випадкових чисел має бути встановлене так, щоб гарантувати унікальну послідовність.

Примітка: використання прапорця -Wall під час компіляції є обов’язковим.

### Програмна реалізація
Для виконання завдання було розроблено програму, яка використовує стандартну функцію rand() для генерації псевдовипадкових чисел. Для забезпечення унікальності послідовності при кожному новому запуску програми було застосовано функцію srand(). Оскільки rand() повертає ціле число, програма приводить його до типу float та нормує шляхом ділення на константу RAND_MAX, що дозволяє отримати значення у діапазоні [0.0; 1.0]. Для отримання числа у діапазоні [0.0; n] отриманий результат додатково множиться на введене користувачем число n.
### Результат виконання
```
gcc -Wall task_7_10.c -o task_7_10
```
**Вивід терміналу:**
```

--- Random Numbers ---

(b) Range [0.0; 108.56]: 75.249619
```
