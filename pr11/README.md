# Практична робота 11

## Завдання 11.4
Реалізуйте систему логування, яка при кожному падінні програми виконує дамп усіх регістрів незалежно від архітектури.
### Програмна реалізація
Для виконання завдання було створено програму, яка перехоплює критичні сигнали аварійного завершення процесу (SIGSEGV, SIGFPE, SIGILL, SIGBUS, SIGABRT) за допомогою  sigaction та записує поточний стан у файл логів.

* Для оптимізації та спрощення коду вивід здійснюється за допомогою функції dprintf(); 
* Незалежність від апаратної архітектури гарантується використанням умовної компіляції препроцесора (#if / #elif / #else);
* Залежно від того, де збирається проєкт (x86_64 чи ARM64), програма звертається до відповідних полів структури ucontext_t;
* Для всіх інших невідомих архітектур передбачено універсальну гілку, що виводить базову діагностику.
### Результат виконання

## Інструкція із запуску
### Підготовка системи
Для виконання роботи вам знадобиться стандартний набір інструментів розробника для мови C.

Встановіть необхідні компоненти у вашій системі:
```
sudo apt update
sudo apt install -y build-essential
```
### Завантаження проекту
```
git clone https://github.com/nanahhr/SystemSoftwareArchitecturePrs.git
cd SystemSoftwareArchitecturePrs/pr11
```

### Компіляція та запуск
Для завдання в директорії підготовлено відповідний вихідний файл (task_11_4.c). Загальна схема запуску виглядає наступним чином:

**Для завдання за варіантом №4:**
```
gcc -Wall task_11_4.c -o task_11_4

./task_11_4
```
